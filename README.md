
A fun and interactive **Quiz Application** built with **HTML**, **CSS**, and **JavaScript** that tests your knowledge with multiple choice questions and tracks your score in real time.

---

## ✨ Features

- ❓ Multiple choice questions
- ✅ Highlights correct and wrong answers instantly
- 📊 Real time score tracking
- 🏁 Final result screen at the end
- 🔄 Restart quiz option

---

## 🎯 Project Purpose

This project was created to practice front-end web development concepts such as:

- 🔄 DOM Manipulation
- 🖱️ Event Handling
- 🧠 Conditional Logic
- 🚀 JavaScript Fundamentals

---

## 🛠️ Technologies Used

- **HTML** – Structure and markup
- **CSS** – Styling and responsive layout
- **JavaScript** – Quiz logic and DOM interaction
